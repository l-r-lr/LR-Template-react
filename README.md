# LR-Template

一个基于 **Vite + React 19 + TypeScript + SCSS + CSS Modules** 的现代化前端项目模板。

![](https://lr-oss-meow.cn-sy1.rains3.com/images/Vector.png)

## 特性

- ⚡ **Vite** - 极速构建，开发体验流畅
- ⚛️ **React 19** - 最新 React 版本，支持新特性
- 🔷 **TypeScript** - 类型安全，开发更安心
- 🎨 **SCSS + CSS Modules** - 样式隔离，模块化开发
- 🌓 **主题切换** - 支持亮色/暗色/跟随系统三种主题
- 📦 **Zustand** - 轻量级状态管理
- 🧩 **组件化设计** - 通用组件开箱即用

## 设计风格

采用 **LR UI 风格** 设计：

- 黑白灰单色设计系统
- 复杂多层打光阴影
- 大圆角、毛玻璃效果
- 无边框悬浮动画

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 运行代码检查
npm run lint
```

## 项目结构

```
src/
├── components/common/    # 通用组件（Button、Card、Modal、Avatar、CodeBlock、Footer）
├── hooks/                # 自定义 Hooks（useToggle）
├── layouts/              # 布局组件（MainLayout）
├── pages/                # 页面组件（Home、About、Docs、NotFound）
├── router/               # 路由配置
├── stores/               # 状态管理（appStore、themeStore）
├── styles/               # 全局样式、主题变量
├── types/                # TypeScript 类型定义
└── assets/               # 静态资源
```
